Manual do painel

O painel é um sistema para controle de perguntas, que contém um cronômetro para programar e iniciar contagens regressivas. A finalidade principal deste sistema é para gerência de tempo e de perguntas para debates de eleições.

O painel possui apenas duas telas: de gerenciador e de usuário.

Na tela de gerenciador, o moderador pode manipular todas as ações: preparar e iniciar cronômetro, mandar mensagens aos usuários participantes, ativar a roleta de números aleatórios, e cadastrar e selecionar perguntas.

Na tela de usuário, é possível visualizar: o cronômetro gerenciado, as mensagens enviadas e os números aleatórios gerados por rodada. Nesta tela, a única ação possível é poder gerar um número de roleta para assim gerar o número de sorteio. Todos os usuários participantes visualizam a mesma tela de usuário.

Criando um novo painel

É possível criar múltiplos paineis para vários debates. Não é necessário autenticação, porém uma vez criado o painel, o criador terá posse de um link exclusivo para gerenciar o painel criado. Este link de gereência não pode ser compartilhado.

  1. Clique aqui para criar um novo painel
  2. Na tela apresentada, os campos já virão preenchidos com valores padrão.
    • O código chave é um valor segredo para poder acessar a página de gerência
    • A opção Ativar regressiva faz com que o cronômetro tenha 2 segundos adicionais para realizar uma contagem regressiva.
  3. Clique em Criar painel;
  4. Na página seguinte, haverá dois links: de usuário e de gerência. Compartilhe o link de usuário para os participantes que visualizarão o painel. NÃO COMPARTILHE o link de gerência! Apenas os moderadores do painel podem acessar o link de gerência.

Tela de gerência

A tela de gerência contém 3 seções:

  • Parte superior: exibe o cronômetro
  • Parte central: gerência
  • Parte inferior: exibe os números de sorteio

A parte central contém os controles do painel: crônometro, mensagem e roleta.

Cronômetro

Na seção Cronômetro, é possível definir o tempo para o cronômetro. O botão Preparar faz mostrar pausado o tempo escolhido na tela para todos os participantes e o botão Iniciar dá início ao cronômetro.

Mensagem pública

Na seção Mensagem pública, é possível enviar mensagens aos usuários participantes. Essa mensagem é visível para todos.

  • A opção Auto-enviar mensagem após a escolha do número sorteado envia o texto da roleta de sorteio automaticamente após a geração do número de sorteio.

Roleta

Na seção Roleta, é possível carregar alternativas para poder sorteá-las. Essas alternativas podem ser perguntas, opções ou qualquer outra coisa que possa ser sorteável.

  • Na caixa de seleção de opções conterá todos os grupos de alternativas que foram carregados. Para carregar um novo grupo, escolha a opção:
    • (novo grupo de sorteio): essa opção sempre estará disponível, e permite carregar um grupo de alternativas. Você precisará de um arquivo de texto puro com todas as alternativas de sorteio (uma alternativa em cada linha). O nome do grupo será o nome do arquivo. Clique no botão Escolher arquivo texto para selecionar e enviar este arquivo.
  • Ativar roleta para os participantes:
    • Ative-a e todos os usuários participantes poderão enviar números aleatórios apenas movimentando o cursor do mouse.
    • Desative-a e o número de sorteio será gerado. Ele é o resultado do resto da divisão da soma de todos os números gerados pelo número de perguntas cadastradas do grupo.
